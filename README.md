# Bicycle-Trip

전국의 자전거길, 축제, 맛집, 카페, 거점, 코스를 비교하고 함께 라이딩하는 국내 자전거여행 큐레이션 정적 웹사이트입니다.

## Pages

- `index.html`: 전국 행사 일정 중심 홈
- `festivals.html`: 전국 행사 일정
- `event.html`: 행사별 상세 안내
- `maps.html`: 카카오맵 기반 코스 지도
- `itineraries.html`: 추천 여행 일정
- `community.html`: 동행 모집, 코스 제안, 준비물, 후기 안내

## GitHub Pages

이 저장소는 GitHub Actions로 정적 파일을 GitHub Pages에 배포하도록 설정되어 있습니다.

1. 변경 사항을 `main` 브랜치에 push합니다.
2. GitHub 저장소의 `Settings > Pages`에서 `Source`를 `GitHub Actions`로 설정합니다.
3. 배포가 끝나면 아래 주소에서 열 수 있습니다.

```text
https://gaja-trip.github.io/Bicycle-Trip/
```

별도 빌드나 패키지 설치는 필요하지 않습니다.
